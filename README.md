# Dual-branch-identity
Machine-learning-enabled dual-branch handwriting digit recognition and user-specific authentication.

The rapid development of flexible electronics provides new opportunities for handwriting-based information interaction and identity verification. Compared with image-based handwriting recognition, pressure-signal-based handwriting recognition records not only the written content but also the individual writing dynamics, including pressing force, writing duration, stroke rhythm, and pressure release behavior. Therefore, the SaLM/SeCNT/PMMA foam pressure sensor can be used as a handwriting-input interface for both digit recognition and user-specific authentication.
